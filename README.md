## **Headings**
Use `#` for headings:
- `#` for H1
- `##` for H2
- `###` for H3
- And so on...

## **Font Styles**
- **Bold text**: `**Bold text**` or `__Bold text__`
- *Italic text*: `*Italic text*` or `_Italic text_`
- ***Bold and italic***: `***Bold and italic***`
- ~~Strikethrough~~: `~~Strikethrough~~`
- `Inline code`: Wrap text with backticks `` `Inline code` ``
